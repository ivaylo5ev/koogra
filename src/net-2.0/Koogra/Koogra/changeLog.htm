Koogra Change Log

3.0 Beta

  • Added Excel 2007 and .Net 3.5 Build
  • Changed indexers on Excel 97 CellCollection and RowCollection classes to Int32.

2.0.2

  • Suppressed exception on unknown URL moniker GUIDs

2.0.1

  • A bug exists when RK values with decimals. Fix was contributed by Jose Flores (foreverjdev)

2.0.0

  • Ported to .Net 2.0

1.1.5

  • A bug exists when reading excel files with a COUNTRY record inserted before a worksheet's INDEX record. This has been corrected. This bug was found by Chris Bell.

1.1.4

  • Incorporated a patch from Chris Bell for the PaletteEntry.GetColor method to enhance a lazy fix to a previous issue.
  • Incorporate a fix by macro187 for a Japanese Character issue found by Jennifer Wen. macro187's fix also takes care of issues related to reading past RTF and Far East character extra information for SST records that are in the continue record.

1.1.3

  • A bug exists when reading floating point RK records. This bug was found by Daniel Rusk. A fix was also proposed by Daniel Rusk and is incorporated in this release.

1.1.2

  • A bug exists when reading BOOLERR values. This bug was also found by Chris Bell. This has been corrected.

1.1.1

  • Shared string table strings with far east data was being read incorrectly. Thanks to Wilson Chan for giving a test file and finding the bug.

1.1.0

  • Added the GetFormattedValue method to the Cell class.
  • When using the path based constructor of the Workbook class, The file may now be opened even if the file is opened by other applications for full read write access. Contributed by Chris Bell.
  • A bug exists in initializing the Palette where some Palette entries could be null. This bug was also found by Chris Bell. Behavior has been set to just stop initializing the Palette if the condition occurs.
  • A condition could occur when loading some compound files where there is a Stream that appears to be broken. The previous behavior would cause the load to fail. This has been modified to continue the load even if such a condition occurs.
  • An NUnit class library has been added for regression testing bugs. This can be enhanced in the future to also test methods but for this release, will be limited to issue regression testing.

1.0.1

  • Changed the Koogra.Worksheet.GetByName method to search for a name in a case insensitive fashion
  • Added the ReadSimpleASCIIString method to the Koogra.Text.Reader class
  • Changed the method ReadSimpleString to ReadSimpleUnicodeString in Koogra.Text.Reader to distinguish between Unicode and Ascii string streams
  • Added support for HLINK records
  • Added support for HyperLink tables
  • Added the HyperLinks property to the Koogra.Excel.Worksheet class which corresponds to each Worksheet's hyperlink table

1.0.0

  • Initial Release
